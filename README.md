# MithunERP - Professional Static Website

## Overview

MithunERP is a modern, responsive, static website built for a professional ERP business. The site features a professional bottle green color theme, fully responsive design, and is optimized for all devices.

## Features

✅ **Professional Design** - Modern and clean interface with bottle green theme  
✅ **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices  
✅ **Mobile Navigation** - Hamburger menu for easy navigation on small screens  
✅ **Modern Color Scheme** - Bottle green with gold accents for professional appeal  
✅ **Complete Sections** - Home, About, Services, Features, Why Choose Us, and Contact  
✅ **Service Showcase** - Display your ERP and software solutions beautifully  
✅ **Contact Form** - Professional contact form with validation  
✅ **Social Media Integration** - Links to social profiles in footer  
✅ **Smooth Animations** - Scroll effects and transitions for better UX  
✅ **Chatbot Ready** - Pre-built placeholder for future chatbot integration  
✅ **SEO Optimized** - Proper meta tags and semantic HTML  
✅ **Accessibility** - WCAG compliant with proper semantic structure

## Color Scheme

The website uses a professional bottle green color palette:

- **Bottle Green**: `#2d5016` (Primary color)
- **Light Green**: `#3d6b1f` (Secondary)
- **Lighter Green**: `#4d8b2f` (Accent)
- **Pale Green**: `#e8f5e9` (Light backgrounds)
- **Dark Green**: `#1b3109` (Dark sections)
- **Accent Gold**: `#d4af37` (Highlights)

## Responsive Design

The website is optimized for:
- **Desktop**: 1200px+ (Full layout)
- **Tablet**: 768px - 1199px (Optimized grid)
- **Mobile**: 480px - 767px (Single column)
- **Small Mobile**: Below 480px (Compact layout)

## Chatbot Integration

The website includes a `#chatbot-container` ready for chatbot integration. When ready to add a chatbot, simply add your chatbot script before the closing `</body>` tag.

## Customization

### Update Contact Information
Edit the contact section in `index.html` to add your actual details.

### Update Colors
Modify CSS variables in `styles.css` to customize the color scheme.

### Add Services
Add more service cards by duplicating the service-card structure.

## Deployment

This is a GitHub Pages repository. Push changes to the main branch and the site will automatically deploy.
